Facial Expression Recognition and Conversion into Emojis

This project detects human facial expressions from images or live video and converts them into corresponding emojis. Using computer vision and deep learning, the system recognizes emotions like happy, sad, angry, surprised, neutral, etc., and displays an emoji that matches the detected expression.
